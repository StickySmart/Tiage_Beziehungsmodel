```
Mystical Osho Zen Tarot card illustration, spiritual meditation art,
soft ethereal lighting, warm earth tones with purple accents,
contemplative atmosphere, sacred symbolism, no text, no words,
digital painting style, centered composition
```

## Tipps für ArtSpace.ai

1. **Batch-Generierung**: Generiere mehrere Bilder pro Session
2. **Prompt Stylizer**: Nutze den eingebauten Stylizer für bessere Ergebnisse
3. **Konsistenz**: Verwende immer den gleichen Basis-Prefix
4. **Auflösung**: Generiere in 1024x1536, dann auf 400x600 verkleinern
5. **Export**: WebP-Format wählen wenn verfügbar

## Nach der Generierung

Die Bilder als `B001.webp` bis `B226.webp` speichern und in:
```
/assets/images/beduerfnisse/
```
ablegen.
