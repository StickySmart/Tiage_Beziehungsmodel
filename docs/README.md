- **A** = Archetyp-Score (0-100) - Beziehungsphilosophie

- **D** = Dominanz-Score (0-100) - Energetische Dynamik

### Resonanz-Faktor

```
R = 0.9 + [(M/100 × 0.35) + (B × 0.35) + (K × 0.30)] × 0.2
```

Wobei:
- **M** = Profil-Match (0-100) - Übereinstimmung der Präferenzen
