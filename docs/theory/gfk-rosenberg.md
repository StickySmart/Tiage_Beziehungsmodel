## Über Marshall B. Rosenberg (1934-2015)

**Marshall B. Rosenberg** war ein amerikanischer Psychologe, der die **Gewaltfreie Kommunikation (GFK)** entwickelte. Sein Ansatz basiert auf der Überzeugung, dass alle Menschen dieselben universellen Bedürfnisse haben und Konflikte entstehen, wenn diese Bedürfnisse nicht erfüllt werden.

**Hauptwerk**: *Gewaltfreie Kommunikation: Eine Sprache des Lebens* (2001)

---

## Das GFK-Modell der Bedürfnisse

### Die vier Schritte der GFK

1. **Beobachtung** - Was nehme ich wahr? (ohne Bewertung)
2. **Gefühl** - Was fühle ich dabei?
3. **Bedürfnis** - Welches Bedürfnis steckt dahinter?
4. **Bitte** - Was wünsche ich mir konkret?

### Anwendung im Tiage-Modell

Das Tiage-Modell fokussiert auf **Schritt 3: die Bedürfnisse**. Diese bilden die Grundlage für die Kompatibilitätsberechnung zwischen zwei Personen.

## Statistische Integration

### Gaußsche Verteilung der Bedürfniswerte

Jeder Bedürfniswert im Tiage-Modell repräsentiert den **Erwartungswert (μ)** einer Normalverteilung:

```
P(μ - 1.28σ ≤ X ≤ μ + 1.28σ) = 0.80

→ 80% der Menschen mit gleicher Merkmalskombination
   liegen in diesem Intervall
```

### Beispiel: "Nähe" für Duo-Submissiv-Cis-Frau

```
μ_archetyp (Duo)     = 95
+ Δ_dominanz (sub)   = +20
+ Δ_geschlecht (cf)  = +20
+ Δ_orientierung     = 0
─────────────────────────
μ_final              = 135 → 100 (Clipping)

Mit σ = 12:
80%-Intervall: [84.6, 100]
```

## Rosenbergs Philosophie im Tiage-Kontext

### Kernprinzipien

1. **Keine Bewertung** - Bedürfnisse sind neutral, nicht "gut" oder "schlecht"
2. **Universalität** - Alle Menschen haben dieselben Grundbedürfnisse
3. **Strategien vs. Bedürfnisse** - Bedürfnisse sind konstant, Strategien variieren
4. **Empathie** - Verstehen der eigenen und fremden Bedürfnisse

### Anwendung im Matching

| GFK-Prinzip | Tiage-Umsetzung |
|-------------|-----------------|
| Keine Bewertung | Score ist keine Qualitätsaussage |
| Universalität | Gleiche 150+ Bedürfnisse für alle Profile |
| Strategien vs. Bedürfnisse | Archetypen = Strategien, Bedürfnisse = Werte |
| Empathie | Kompatibilität = überlappende Bedürfnisse |

## Kritische Würdigung

### Stärken der GFK-Basis

- **Wissenschaftlich fundiert** - Jahrzehnte der Forschung und Anwendung
- **Kulturübergreifend** - Universelle Bedürfnisse
- **Praxisorientiert** - Direkte Anwendung in Konfliktsituationen
- **Nicht-pathologisierend** - Keine "falschen" Bedürfnisse

### Limitationen

- **Westliche Perspektive** - Entwickelt in westlichem Kontext
- **Individualfokus** - Kollektivistische Kulturen unterrepräsentiert
- **Dynamik-Erweiterung** - BDSM-Kategorie nicht von Rosenberg
- **Quantifizierung** - Rosenberg arbeitete qualitativ, nicht mit Scores
